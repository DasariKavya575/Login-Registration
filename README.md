# Login-Registration
"Created a Login and Registration System using Python, Streamlit, and SQLite. Users can sign up, log in, and reset passwords. All details are securely stored in SQLite, and Streamlit provides a clean, user-friendly interface for Login, SignUp, and Forgot Password pages."

## Features

- **User Registration:** Create a new account with a unique username and password.  
- **User Login:** Log in using registered credentials.  
- **Forgot Password:** Reset password if forgotten.  
- **Secure Storage:** All user data stored in an SQLite database.  
- **Clean Interface:** Easy-to-use pages built with Streamlit.

## Technologies Used

- **Python** – Main programming language  
- **Streamlit** – For interactive web interface  
- **SQLite** – Database to store user information
